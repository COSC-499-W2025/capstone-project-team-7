# User Task List

## Getting Started

1. **Sign up for a new account** using your email and password
2. **Log in** with your credentials
3. **Accept the consent agreement** to enable AI-powered features
4. **Select scanning preferences** on User Configuration

## Scanning & Exploring

1. **Run a portfolio scan** on a project folder or zip file
2. **Find how many files were scanned** in your project
3. **View the language breakdown** to see which languages are used most
4. **Search for a specific file** by name in the scan results
5. **Find duplicate files** in your project and check how much space they waste

## Projects & Storage

1. **Save your scan as a project** to keep it for later
2. **View your saved projects list** and find the most recent one


## AI Analysis & Resume

1. **Run AI-powered analysis** and view the generated portfolio summary
2. **Generate a resume item** from your scan results
3. **View skill progression**

## Wrap Up

1. **Log out** and confirm your session ends cleanly
